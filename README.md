# WorldChess QA Agent

Это проект AI-агента для автоматизации QA-пайплайна WorldChess.

## Структура проекта

- `check-lists/` — входная папка для файлов чеклистов.
- `prompts/` — сгенерированные промпты для LLM.
- `results/` — результаты шагов: тест-кейсы, код-ревью, баг-репорты.
- `autotests/` — сгенерированные TypeScript-автотесты.
- `src/agent.ts` — точка входа агента.
- `src/openai.ts` — хелпер для вызова OpenAI API.

## Настройка

1. Установите зависимости:

```bash
npm install
```

2. Установите переменную окружения:

```bash
setx OPENAI_API_KEY "ваш_ключ"
```

Или запустите команду напрямую:

```bash
$env:OPENAI_API_KEY = "ваш_ключ"
```

## Запуск агента

```bash
npm run dev
```

Или после сборки:

```bash
npm run build
npm start
```

## Как это работает

Агент выполняет следующие шаги:

1. Загружает файл чеклиста из `check-lists/`.
2. Генерирует формальные тест-кейсы и сохраняет их в `results/test_cases.json`.
3. Создаёт TypeScript автотесты и сохраняет их в `autotests/worldchess.spec.ts`.
4. Пропускает выполнение автотестов.
5. Делает код-ревью и сохраняет отчет в `results/code_review.md`.
6. Генерирует баг-репорты и сохраняет их в `results/bug_reports.json`.

## Примечания

- Проект ожидает, что в папке `check-lists/` уже есть файл с чеклистом.
- Все вызовы LLM осуществляются через OpenAI API.
- В генерации промптов используется формат Markdown и JSON.
